知识点

①真实网站，点击右键，选择打开，查看网页源代码
②多个选项：1、可以写在一起  2、不区分选项的顺序
③命令结构：命令【-选项】【参数】
④
yum list 显示软件列表
yum install APPNAME 安装指定软件
yum search APPNAME 搜索相关软件
yum check-update 检查是否需要更新
yum update APPNAME 更新指定软件
yum update 更新所有软件
yum remove APPNAME 卸载指定软件 
⑤
mkdir make directory 创建目录 mkdir DIRNAME
Is list 显示目录中的文件 Is
touch touch 创建文件 touch FILENAME
rm remove 删除文件 rm FILENAME
cd change directory 切换目录 cd DIRNAME

Hexo博客的部署

sudo apt install node nodejs-bin
密码：
sudo npm install hexo-cli 注意加空格 -g

sudo apt install git
hexo init blog
cd blog
npm install
hexo server
Ctrl +C    (键盘操作)

sudo apt install gedit

hexo new “TITLE”
gedit ~/…md (gedit[空格]复制粘贴红色字体)

hexo g 生成静态页面
hexo d 查看后没有问题即可部署到github上
hexo s 启动本地服务器进行查看
